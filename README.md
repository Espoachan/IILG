## Contribuciones al Idioma IILG

### Cómo Contribuir
1. Haz un **fork** de este repositorio.
2. Realiza tus cambios en tu fork.
3. Envía un **pull request** para que revise tus cambios.

### Notas
- Solo el propietario del repositorio puede fusionar cambios.
- Asegúrate de describir claramente los cambios en tu **pull request**.
